How to Organize and Analyze Research Data

This short lesson is intended for graduate and advanced undergraduate students undertaking original research projects. After completing the 30-minute lesson, you will be able to:

  • Format raw research data in an Excel spreadsheet for efficient analysis
  • Run basic descriptive statistics in Excel
  • Prepare the document for statistical analysis in the software package PASW

Requirements

For this lesson, you'll need the following:

  • Microsoft Excel
  • PASW (formerly SPSS) statistics software, or free trial download

You should already be familiar with:

  • The basic functions and tools of Excel
  • Descriptive statistics and inferential statistics
  • Research design and how to identify independent variables and dependent variables

Formatting Data for Analysis

Ensure that the computer you are working on has Microsoft Excel installed. Open the document ToolBox_DataLesson.xls and follow along.

  1. In the Raw_Dataset spreadsheet, enter variable names in the first row of each column. Make each name unique and ensure each variable name begins with a letter.
  2. Rename variables for clarity, such as changing use to perc_ease_of_use and useful to perc_usefulness.

Tip: Dichotomous Variables

Dichotomous variables allow the easy isolation of characteristics during analysis. For instance, separating experience levels lets you test in PASW if subjects with low experience have scores that significantly differ from those with higher experience levels.

  1. In the Formatted_Data sheet, copy the Raw_Dataset values and separate each distinct value into a separate column.

Running Descriptive Statistics

Analyze descriptive statistics to start appraising your dataset. Here’s how to calculate descriptive statistics in Excel:

  1. Write down the input range of the data to summarize (e.g., H2:J26).
  2. Ensure that the Analysis ToolPak is enabled under Tools > Add-Ins.
  3. Under Tools > Data Analysis, select Descriptive Statistics and check Summary Statistics.

Analyzing Descriptive Statistics

Descriptive statistics provide insights into the dataset, such as identifying segments that may not find a system useful despite enjoying its use.

Running Inferential Statistics in PASW

Open PASW and upload the Excel spreadsheet. Select Formatted_Data as the data source. You can now access common statistical tests in PASW under Analyze.

Choosing the Correct Statistic

Consult UCLA’s Choosing the Correct Statistic resource to ensure the correct statistical test for your research question.

Example: Kruskal-Wallis Test

To determine if there is a significant difference in perc_usefulness scores between free_play and other tutorial types, use the Kruskal-Wallis test:

  1. Select Analyze > Nonparametric Tests > K Independent Samples...
  2. Set perc_usefulness as the Test Variable and free_play as the Grouping Variable.

Conclusion

By following this lesson, you have organized and analyzed research data using Excel and PASW. Completing the Quiz will help reinforce your understanding of formatting datasets for statistical analysis.
